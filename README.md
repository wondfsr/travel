## 프로젝트 소개
- React를 사용해 개발한 여행 일정 사이트의 프론트엔드입니다.
- 여러 api들을 사용하여 데이터를 활용하였습니다.

## 주요 기능
- 여행지 선택
- 여행 장소별 상세 설명
- 일정에 장소 추가 및 순서 변경
- 이동 시간 표시
- 지도에 일정에 추가한 장소 및 경로 표시
